## Notes

### Create React App

`$ npm create vite@latest [vite-app|.]`
`$ cd vite-app`
`$ npm install`
`$ npm run dev`

### Install SASS

`$ npm i -D sass`

### Build

`$ npm run build`

### Run Preview

`$ npm run preview`
